# Crawl Original Google Images

---

### Overview

Download original (not thumbnails) from Google Images Search with **multi-threading** :D
1. Get URLs by keywords

```
	python crawl_url.py
```

1. Download imgs from URLs

```
	python crawl_data.py
```
